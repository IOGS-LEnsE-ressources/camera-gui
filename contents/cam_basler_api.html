CameraList class

class camera_list.CameraList

Class to list Basler camera (all camera in a future evoluation)

Parameters:

  • available_cameras (list[tuple[int, str, str]]) – List of the devices connected to the computer [id, device]

  • camera_list_str – List to print of the devices connected to the computer [id, serial number, device name]

  • nb_cam (int) – Number of connected cameras

Example

In this example, you create a CameraList object, then get the number of Basler cameras connected to your computer (in this case 1 camera is connected) and finally obtain the complete list of the available cameras (in this case a a2A1920-160ucBAS camera with the serial number 40282239).

>>> from camera_list import CameraList
>>> cam_list = CameraList()
>>> cam_list.get_nb_of_cam()
'1'
>>> cameras_list = cam_list.get_cam_list()
>>> print(cameras_list)
'[[0, 40282239, 'a2A1920-160ucBAS']]'

CameraBaslerWidget class

class camera_basler_widget.CameraBaslerWidget(*args: Any, **kwargs: Any)

CameraBaslerWidget class, children of QWidget.

Class to integrate a Basler camera into a PyQt6 graphical interface.

Parameters:

  • cameras_list_widget (CameraBaslerListWidget) – Widget containing a ComboBox with the list of available cameras.

  • main_layout (QGridLayout) – Main layout container of the widget.

  • camera (pylon.TlFactory) – Device to control

Note

The camera is initialized with the following parameters :

  • Exposure time = 100 ms

  • FPS = 5

  • Black Level = 0

  • Color Mode = ‘Mono12’ (if possible)

Parameters:

  • camera_display (QLabel) – Area to display the camera image

  • camera_infos (SmallParamsDisplay) – Area to display camera informations (FPS, expotime…)

  • main_timer (QTimer) – timer object to manage display refresh

clear_layout() → None

Clear the main layout of the Widget.

Note

This function is used to display camera image instead of the camera list when a camera is selected and the user clicks on the connect button.
